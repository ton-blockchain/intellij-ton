Reports if and ifnot statements that contain only a single throw call in their body. Such constructs can be converted to the more idiomatic throw_if or throw_unless functions.

Examples:

Before: 

if (i > b) {
  throw(error);
}

ifnot (condition) {
  throw(error);
}
After: 

throw_if(error, i > b);

throw_unless(error, condition);
The throw_if and throw_unless functions are part of the FunC standard library and are syntactically more correct and idiomatic for conditional error throwing.
