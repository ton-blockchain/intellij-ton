Reports functions that return constant values using assembly and can be converted to idiomatic constant declarations.

Functions that use asm with PUSHINT instruction to return constant integer values can be replaced with more readable and efficient const declarations.

Example:

Before: 

int min_tons_for_storage() asm "10000000 PUSHINT";
After: 

const min_tons_for_storage = 10000000;

Using const declarations instead of assembly functions for constants improves code readability, reduces gas costs, and makes the code more maintainable. The compiler can also optimize constant usage better.
